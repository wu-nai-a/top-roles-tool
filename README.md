# AI 主播替换工具 — Top角色数据分析平台

> 管理后台直播绑定平台 · 智能角色筛选 & 数据分析

## 功能概览

上传 XLSX 角色数据表，自动完成以下处理：

### 📊 数据清洗
- 自动识别并修复文本格式的数字列（总聊天人数、user_id 等）
- 按性别（男/女）自动分类
- 按总聊天人数降序排序

### 🔍 多维度筛选
- **全部角色** — 总表概览
- **男性 / 女性** — 分性别排行
- **新音色** — 筛选使用新版音色 ID（如 `nansheng_3`、`nvsheng_6` 格式）的角色
- **新星 / 沉淀** — 根据创建时间区分近期热门角色与长线热门角色
- **分页浏览** — 筛选结果每页显示 500 条，可继续查看 501-1000 等后续数据

### 📈 数据分析看板
- **热门标签分析** — 统计每个标签出现频次、平均聊天人数、性别分布，附带中文释义
- **热门题材提取** — 从角色简介中自动提取核心题材（如"京圈太子爷"、"豪门富少"）
- **性别偏好对比** — 男/女角色热门题材分列对比
- **图表辅助判断** — 分析摘要、热门题材、热门标签、性别偏好均配套图表展示

### 📋 替换清单导出
- 勾选需要上线的角色，添加至替换清单
- 可自定义导出字段，默认导出 `user_id` 和头像 URL
- 一键导出为结构化 XLSX 文件
- 支持快捷键 `Cmd/Ctrl + S` 快速导出

### 🖼️ 头像预览
- 表格头像支持点击放大查看
- 支持点击空白区域或按 `Esc` 关闭预览

### 🧭 首次使用引导
- 新用户首次打开页面会看到快速上手弹窗
- 页面右上角可随时重新打开“使用说明”

### 🖼️ 角色状态标注
每个角色自动标注状态标签：
- **🎙️新音** — 使用新版音色 ID 的角色
- **⭐新星** — 近期创建且数据表现突出的角色
- **💎沉淀** — 创建时间久、用户积累深厚的热门角色

## 使用说明

1. **打开页面** — 直接在浏览器中打开 `index.html`（纯前端，无需服务器）
2. **上传文件** — 点击上传区域或拖拽 XLSX 文件
3. **等待处理** — 处理进度条会显示当前阶段
4. **浏览数据** — 通过顶部 Tab 切换不同分类表格，使用分页查看后续角色
5. **分析数据** — 分析板块会自动生成表格和图表
6. **导出清单** — 勾选角色 → 点击"添加至替换清单" → 选择导出字段 → 导出

### 注意事项
- 推荐上传 **200MB 以内** 的 XLSX 文件；200MB 以上会提示性能风险，500MB 以上会阻止上传
- 所有数据处理在**浏览器本地**完成，不上传任何数据到服务器
- 推荐使用 Chrome / Edge 等现代浏览器

## 部署方式

### 方式一：GitHub Pages（推荐）

1. 在 GitHub 上创建一个新仓库
2. 将本目录所有文件推送至仓库的 `main` 分支
3. 进入仓库 Settings → Pages → 选择 `main` 分支，根目录
4. 等待几分钟，即可通过 `https://<你的用户名>.github.io/<仓库名>/` 访问

### 方式二：直接分享

将 `index.html` 文件发送给同事，直接用浏览器打开即可使用（功能完整）。

### 方式三：任意静态托管

上传至任何支持静态文件的托管服务（如 Vercel、Netlify 等）即可。

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- [SheetJS (xlsx)](https://sheetjs.com/) — Excel 文件解析与导出
- 无后端依赖，无服务器成本
